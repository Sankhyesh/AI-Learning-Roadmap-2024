2. Load Cards from URL Parameter

You can pass a JSON array of cards as a URL parameter:

index.html?cards=[{"front":"Question 1","back":"Answer 1"},{"front":"Question 2","back":"Answer 2"}]

3. Generate Links Programmatically

Create a link in your own HTML/JS application:

const cards = [
  {
    front: "What is LangChain?",
    back: "An open-source framework for LLM applications",
    type: "basic",
    tags: ["LangChain", "LLM"]
  },
  // more cards...
];

const url = `index.html?cards=${encodeURIComponent(JSON.stringify(cards))}`;
// Use this URL in a link or window.location.href

4. Load Cards from External JSON

You can also load cards from an external JSON file using JavaScript:

// In your HTML file
<button onclick="loadExternalCards('path/to/your/cards.json')">
  Load External Cards
</button>

<script>
async function loadExternalCards(url) {
  try {
    const response = await fetch(url);
    const cards = await response.json();
    const appUrl = `index.html?cards=${encodeURIComponent(JSON.stringify(cards))}`;
    window.location.href = appUrl;
  } catch (error) {
    console.error('Error loading external cards:', error);
    alert('Failed to load external cards');
  }
}
</script>
